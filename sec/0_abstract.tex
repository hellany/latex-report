\begin{abstract}
Facial Expression Recognition enables machines to infer human emotions from facial images. This report explores DL approaches for classifying low-resolution (64×64) facial images into six basic emotion categories. We review relevant literature, compare CNN-, transformer-, and hybrid-based architectures, and outline data preprocessing, augmentation, and class imbalance handling strategies. The goal is to identify an accurate, efficient, and interpretable FER model suitable for training from scratch. 
\end{abstract}
